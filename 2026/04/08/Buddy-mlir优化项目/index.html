将AffineMap转为vector

添加头文件

在匿名空间内添加函数

AffineMap permutation: memref::transpose的permutation本质上是affine map 一个线性的关系

values.reserve(permutation.getNumResults());: 获取affine map大小 提前为values扩容

e.g:(d0, d1, d2) -> (d2, d0, d1) 那么就是 reserve(3)

for (AffineExpr expr : permutation.getResults()): 遍历affine map的每一个结果表达式

例如:(d0, d1, d2) -> (d2, d0, d1) 取出来就是d2,d0,d1

values.push_back(cast<AffineDimExpr>(expr).getPositon());: 将维度强行转换为数字

整个函数效果就是:(d0, d1, d2) -> (d2, d0, d1) 转为[2,0,1]

compose函数

函数实现的就是公式:compose(perm1,perm2) = perm1[perm2[i]]

也就是perm1经过perm2的转换后得到的

e.g:

perm1 = [1, 0, 2]
perm2 = [2, 0, 1]

compose(perm1, perm2) = [2, 1, 0]

strided<[...]> 表示 每个维度在内存中移动一步要跨多少元素（步长）

第二个Pass编写

copy elimination 解决冗余数据搬运消除

要看block内前后操作关系 判断中间操作是否安全 保守处理side effect

至少满足

  1. 第一条 copy 确实是 a -> b
  2. 第二条 copy 确实是 b -> a
  3. 它们在同一个 block 内
  4. 中间没有对 %a 或 %b 的写
  5. 中间没有未知副作用 op
  6. 中间可以有：
    • 对 %a / %b 的只读操作
    • 对其他 buffer 的读写
    • 与 %a/%b 无关的纯计算
