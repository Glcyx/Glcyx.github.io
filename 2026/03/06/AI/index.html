深度学习入门

发表于 2026-03-06 更新于 2026-04-20 分类于 笔记

目标:

了解深度学习算法基本原理
指你要懂常见深度学习算法最核心的思想，例如：

  • 前向/反向传播
  • 梯度下降及其变种（SGD、Adam 等）
  • 卷积、池化、注意力、RNN 等基本机制
    不是要求你把数学推导到极致，但要能说明它们为何有效、怎么工作的。

熟悉神经网络基本架构及其算子计算方式

  • 基本架构：CNN、RNN、Transformer、MLP 等
  • 算子计算方式：卷积算子怎么计算、矩阵乘法如何参与前向传播、激活函数怎么处理、LayerNorm/BatchNorm怎样运作等
    重点是理解每个算子如何接收输入、如何计算、如何输出。

了解至少一种深度学习训练框架及其模型文件解析（如 PyTorch、TensorFlow）

  • 会用 PyTorch 或 TensorFlow 做训练、推理
  • 知道模型文件（如 .pt、.pth、.pb、saved_model）里面大概包含什么：
    权重参数、模型结构、图信息等
  • 能够基本读取、保存、加载模型
  • 有时还要求你能解析模型计算图或理解 ONNX 导出流程。

机器学习入门

有监督学习:

大致分为两类: 回归任务.分类任务

回归: 结果是不确定的无限连续值, 比如:模型输出猫的体重是1.1kg,可能是1.2,1.3…..理论上精度无限,所以连续

分类: 分类任务结果种类是固定的,比如说 输出结果是猪狗猫,模型会告诉你是这三个中的一种,不可能是牛

线性回归模型(回归任务，预测连续值)

w权重,b偏置

误差(输出与真实值之间的差距)计算 设定误差函数

y =wx+b y:特征 w: 标签

  1. 穷举法

  2. 最小二乘法,

  3. 梯度下降法 学习率(a), 三个中最优算法

数据是否带标签然后输入到模型训练是有无监督学习的区别,有监督学习带标签

逻辑回归模型(分类任务，预测类别概率)

逻辑回归算法是二分类, 可以理解为利用回归的方法进行分类,

sigmoid函数:
$$
\sigma(x)=z = \frac{1}{1 + e^{-x}}
$$
导数:
$$
\sigma’(x) = \sigma(x) \left(1 - \sigma(x)\right)
$$

我们将x转化为输入的数据特征,这个时候可以利用sigmoid函数对其进行映射,

z=w0x0+w1x1+…+wnxn + b

此时根据z值 靠近1或0 得到二分类结果,

评价指标:

分类模型评价指标: 准确率,精确率,召回率,F1值

回归模型评价指标: 平均绝对误差,均方误差(loss值),均方根误差,MAPE(平均绝对百分比误差) 都大差不差

深度学习入门

全连接神经网络(fully connected neural network,FCNN或MLP:multilayer perception)

最基础的神经网络结构,CNN,RNN,transformer可以看作其进化版

整体结构:
输入层-> 隐藏层->输出层

神经元的数学表达式:

$$
a = h(wx + b)
$$

h是激活函数,激活函数让神经网络从线性变成非线性,打破每一层之间的线性关系,赋予网络非线性能力

例如前面所学的sigmoid函数就是一个激活函数

每个神经元对应一个b,每个输入的x 对应一个参数w,

所以这一层参数个数就是(x的数量乘上神经元数量) 再加上神经元的数量

激活函数

sigmoid优点: 简单 非常适用分类任务

Tanh函数:
$$
\tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}}
$$
导数

$$
\tanh’(z) = 1 - \tanh^2(z)
$$
ReLU函数:
$$
f(x) = \begin{cases} 0 & \text{if } x > 0 \ x & \text{if } x \le 0 \end{cases}
$$

$$
\frac{d}{dx}\operatorname{ReLU}(x)=\begin{cases}1,&x>0\0,&x\le0\end{cases}
$$

优点: 解决了梯度消失(>0情况下) ,计算更为简单

缺点: 训练时可能出现神经元死亡(x<0,导数为0,w和b不更新了)

Leaky ReLU函数:
$$
f(x) = \begin{cases} 0 & \text{if } x > 0 \ ax & \text{if } x \le 0 \end{cases}
$$
解决了神经元死亡问题

前向传播

从输入开始，经过一层层加权计算、非线性变换，最终得到输出结果

也就是y=model(x) ,输入x得到y的过程

神经网络的损失函数 loss

… 有多种衡量标准

比如;

f(x) 就是关于w,b的函数嘛

反向传播

案例

利用损失函数,通过链式法则，从输出层向前一层层计算每个参数的梯度，使得我们能知道如何更新参数以降低 loss。

梯度下降法是反向传播具体的一个实现过程 利用损失函数的导数来更新w,b

所以前面导数为0,就会出现神经元死亡,w,b都不更新了怎么整

梯度下降法参数更新公式:

a就是学习率,根据其大小,调整w跨度,大了可能错失最小w,小了时间资源花费大

本身w的导数就是一个二次函数,其存在最小值 我们的目的就是尽可能地逐步更新w 然后找到他,这样 损失值就越来越小,与真实值差距就减少了,模型是不是就更准确了啊

b同理

当然这里为什么是减去a*导数,因为导数是正的话,向右走,我们得减去他才能往山沟里(最低点)走啊,如果导数是负的,那就在中心线左边啊,那就是往左边走了,减去他才往右走

过拟合:

模型的复杂度太高（参数太多或结构太灵活），导致它在训练数据上学习到了随机误差，而不是数据真正的统计规律。

常见解决办法包括：

  • 减少模型复杂度（比如浅一点的神经网络、少一点参数）
  • 正则化（Regularization），对模型参数加约束
  • 数据增强或增加训练样本
  • 使用交叉验证选择合适的超参数

卷积神经网络(Convolutional Neural Network,CNN)

全连接神经网络在图像数据输入时会破坏图像的特征,因为数据是一条一条的输入

卷积运算后特征图大小:

H:高,W:宽,P:填充,FH:卷积核高,FW:… OH: 特征图高,OW:…

利用立体图来表示卷积多通道运算:

卷积核的通道与输入通道数量相同(C), 输出特征图通道与卷积核数量相同(FN)

池化操作

压缩空间维度

感受野: 类似于卷积核

最大池化,平均池化 等..

不会改变特征图的通道数,除非特殊操作

LeNet(LeNet-5)

5代表这个网络里一共有五个含有参数的层

这里两个卷积层,三个全连接层

Flatten : 展平操作,把图形展平为一条数据,方便与全连接层相连

组合使用卷积层,sigmoid函数和全连接层

两块组合,然后展平交给全连接层,用来做分类或者回归

AlexNet

使用ReLU 作为激活函数替换了sigmoid或tanh函数,加快收敛,减少训练时间

池化操作从平均池化改为最大池化

softmax:

$$
\text{Softmax}(x_{i}) = \frac{\exp(x_i)}{\sum_j \exp(x_j)}
$$

softmax 把一组任意实数转成一组像概率的数，让模型在多选一的场景里做出选择。

后面全连接层的神经元个数4096等都是这个模型创建者自己设立的,你别管为什么是4096 反正他跟你说这个叼

当然最后一层看你最后分类数量,你要分类1000个那就是1000个神经元,你做二分类,那就是两个,你要识别数字,那就是10个

Dropout:

流程：

(1)首先随机（临时）删掉网络中一半的隐藏 神经元，输入输出神经元保持不变（图中虚线为 部分临时被删除的神经元）

(2) 然后把输入x通过修改后的网络前向传播， 然后把得到的损失结果通过修改的网络反向传播。 一小批训练样本执行完这个过程后，在没有被删 除的神经元上按照随机梯度下降法更新对应的参 数（w，b）。

(3) 然后继续重复这一过程

  1. 加速训练
  2. 防止某个参数过度训练(导致这个参数占比很高或低),减少过拟合

图像增强:(增加图像数量,扩充数据集)

  1. 水平翻转
  2. 随即裁剪
  3. PCA

LRN正则化(局部归一化):

n为所选神经元前后通道的个数,没有则该值设为0
